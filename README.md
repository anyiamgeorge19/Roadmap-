# Full Stack Web Development Roadmap (Free Resources)

Estimated time: 6-12 months at 10-15 hrs/week, depending on prior experience.

---

## Phase 0: Foundations (1-2 weeks)
- **How the web works** — [MDN: How the Web Works](https://developer.mozilla.org/en-US/docs/Learn/Getting_started_with_the_web/How_the_Web_works)
- **Command line basics** — [freeCodeCamp: Command Line for Beginners](https://www.freecodecamp.org/news/command-line-for-beginners/)
- **Git & GitHub** — [Git & GitHub Crash Course (freeCodeCamp)](https://www.freecodecamp.org/news/git-and-github-for-beginners/), [GitHub Skills (interactive)](https://skills.github.com/)

---

## Phase 1: HTML, CSS, JS Basics (4-6 weeks)
- **HTML & CSS** — [freeCodeCamp Responsive Web Design](https://www.freecodecamp.org/learn/2022/responsive-web-design/)
- **CSS deep dive** — [CSS Flexbox](https://flexbox.zeoob.com/) or [Flexbox Froggy](https://flexboxfroggy.com/) + [Grid Garden](https://cssgridgarden.com/)
- **JavaScript fundamentals** — [javascript.info](https://javascript.info/) (best free JS reference, start to finish)
- **JS practice** — [freeCodeCamp JavaScript Algorithms and Data Structures](https://www.freecodecamp.org/learn/javascript-algorithms-and-data-structures/)

**Project checkpoint:** Build 2-3 static sites (portfolio, landing page, calculator).

---

## Phase 2: JavaScript Deep Dive + DOM (3-4 weeks)
- **DOM manipulation** — [The Odin Project: Foundations](https://www.theodinproject.com/paths/foundations/courses/foundations)
- **Async JS (promises, fetch, async/await)** — [javascript.info: Promises, async/await](https://javascript.info/async)
- **ES6+ features** — [freeCodeCamp ES6 tutorial](https://www.freecodecamp.org/news/javascript-es6-tutorial/)

**Project checkpoint:** Weather app, to-do list, quiz app using APIs.

---

## Phase 3: Version Control & Tooling (ongoing, ~1 week upfront)
- **Git branching, PRs, workflows** — [Learn Git Branching (interactive)](https://learngitbranching.js.org/)
- **npm/package managers basics** — [npm docs](https://docs.npmjs.com/)

---

## Phase 4: Frontend Framework — React (4-6 weeks)
- **Official docs (excellent, free)** — [react.dev](https://react.dev/learn)
- **Full course** — [The Odin Project: React path](https://www.theodinproject.com/paths/full-stack-javascript/courses/react)
- **Alternative full course** — [freeCodeCamp: React Course](https://www.freecodecamp.org/news/free-react-course-2022/)

**Project checkpoint:** Build a multi-page React app consuming a public API (e.g., movie search, recipe finder).

---

## Phase 5: Backend Development (5-7 weeks)
- **Node.js basics** — [Node.js official docs](https://nodejs.org/en/docs)
- **Express.js** — [Express official guide](https://expressjs.com/en/starter/installing.html)
- **REST API design** — [freeCodeCamp: REST APIs](https://www.freecodecamp.org/news/rest-api-design-best-practices/)
- **Full backend course** — [The Odin Project: NodeJS course](https://www.theodinproject.com/paths/full-stack-javascript/courses/nodejs)

**Project checkpoint:** Build a REST API (CRUD) for a blog/notes app.

---

## Phase 6: Databases (3-4 weeks)
- **SQL fundamentals** — [freeCodeCamp: SQL course](https://www.freecodecamp.org/learn/relational-database/)
- **PostgreSQL** — [PostgreSQL Tutorial](https://www.postgresqltutorial.com/)
- **NoSQL / MongoDB** — [MongoDB University (free courses)](https://learn.mongodb.com/)
- **ORM (Prisma or Mongoose)** — [Prisma docs](https://www.prisma.io/docs) or [Mongoose docs](https://mongoosejs.com/docs/guide.html)

**Project checkpoint:** Connect your Express API to a real database (Postgres or MongoDB).

---

## Phase 7: Full Stack Integration + Auth (3-4 weeks)
- **Authentication (JWT, sessions)** — [freeCodeCamp: Authentication tutorial](https://www.freecodecamp.org/news/how-to-authenticate-users-in-express-js/)
- **Full stack MERN course** — [freeCodeCamp: MERN Stack](https://www.freecodecamp.org/news/create-a-mern-stack-app/)

**Project checkpoint:** Full stack app with user login, protected routes, and CRUD (e.g., task manager, e-commerce cart).

---

## Phase 8: Deployment & DevOps Basics (1-2 weeks)
- **Frontend hosting** — [Vercel docs](https://vercel.com/docs) (free tier)
- **Backend hosting** — [Render docs](https://render.com/docs) or [Railway docs](https://docs.railway.app/) (free tiers)
- **Docker basics (optional but valuable)** — [Docker: Get Started](https://docs.docker.com/get-started/)

---

## Phase 9: Computer Science Fundamentals (parallel track, ongoing)
- **CS50 (Harvard, free)** — [cs50.harvard.edu/x](https://cs50.harvard.edu/x/)
- **Data Structures & Algorithms** — [NeetCode.io](https://neetcode.io/) + [NeetCode 150 list](https://neetcode.io/practice)
- **Practice problems** — [LeetCode (free tier)](https://leetcode.com/)

---

## Phase 10: Job Readiness (3-4 weeks)
- **Build 2-3 polished portfolio projects** showcasing full stack skills (not tutorials — original ideas)
- **Resume + GitHub profile polish** — [roadmap.sh: Full Stack Roadmap](https://roadmap.sh/full-stack) for a visual skill checklist
- **Mock interviews / behavioral prep** — [freeCodeCamp: Interview Prep](https://www.freecodecamp.org/news/tag/interview-prep/)
- **System design basics (for later-stage interviews)** — [github.com/donnemartin/system-design-primer](https://github.com/donnemartin/system-design-primer)

---

## Suggested Capstone Projects (pick 2-3 for your portfolio)
1. Full stack e-commerce site (auth, cart, payments via Stripe test mode)
2. Social media / forum clone (posts, comments, likes, auth)
3. Real-time chat app (Socket.io + Node + React)
4. Project management tool (like a mini Trello) with drag-and-drop

---

## Notes
- Don't skip fundamentals for frameworks — employers test vanilla JS/HTML/CSS knowledge often.
- Ship projects publicly on GitHub with good READMEs; consistency matters more than speed.
- [roadmap.sh](https://roadmap.sh/) has visual, up-to-date roadmaps for full stack, frontend, and backend if you want a second reference alongside this one.
